Bootstrap Features Showcase

1. Navbar:

Bootstrap's styling for navigation pills, such as `nav-pills` used in the "Who am I?", "Education", and "Contact Me" sections

2. Carousel:

Entire carousel section implemented using Bootstrap's carousel component such as `carousel` class

3. Buttons:

"Contact Me" button styled using Bootstrap's button classes such as `btn` and `btn-primary`

4. Modal:

Modal triggered by the "Contact Me" button created using Bootstrap's modal component

5. Grid System:

Layout structured using Bootstrap's grid system with classes like `container-fluid`

6. Tables:

Tables styled using Bootstrap's table classes such as `table`, `table-dark`, `table-striped`

7. Cards:

Project cards in the "My University Projects" section implemented using Bootstrap's card component such as `card` class

8. Card Groups:

Card group in the "Ideal Jobs" section created using Bootstrap's card group component

9. Utilities:

Utility classes used for text alignment `text-center`, image styling `img-fluid`, `rounded`, and background colors `container-bg`, `container-bg2`
